# Beyzanur ÇİMEN - İlk Web Sayfamız

Bu depo, "Ödev 1 - İlk Web Sayfamızı Oluşturmak" ödevi için hazırlanmış tek sayfalık bir HTML projesidir. Sayfa, HTML5 yapısı ve dahili CSS kullanılarak oluşturulmuştur ve ödevde istenen tüm öğeleri içerir.

Özet
- Dosya: `index.html` — Başlık, paragraflar, bölüm başlıkları, sıralı (`<ol>`) ve sırasız (`<ul>`) listeler, resim (`<img>`), bağlantılar (`<a>`), iframe (OpenStreetMap harita) ve dahili stil (`<style>`) içerir.
- README dosyası proje açıklaması ve çalıştırma bilgilerini içerir.

Özellikler
- HTML5 standartlarına uygun yapı
- Bölümlere ayrılmış CV/özgeçmiş içeriği (Özgeçmiş, Eğitim, İş Deneyimi, Yetenekler, İletişim)
- En az bir sıralı ve bir sırasız liste (her biri 3+ madde)
- Placeholder resim (kolayca yerel fotoğrafla değiştirilebilir)
- OpenStreetMap üzerinden Atatürk Üniversitesi (Erzurum) konumu embed edilmiş iframe
- Basit, responsive uyumlu CSS (mobil görünüm için medya sorgusu)

Klasör yapısı
```
ödev4/
├─ index.html      # Ana sayfa
└─ README.md       # Bu dosya
```
